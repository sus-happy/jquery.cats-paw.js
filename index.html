関数説明

ロールオーバー ( rollOver )

画像のロールオーバーを設定する関数

関数概要

{target}.rollOver({option});
{target}
適用する対象
{option}
[find](string) = null
{target}の絞り込み
[off](string) = '_off.'
ロールオーバー前画像の末尾語
[on](string) = '_on.'
ロールオーバー後画像の末尾語

デモ

ファイル名の最後が_offで終わっている画像を全てロールオーバーする

$("img[src*='_off.']").rollOver();
input要素の画像をロールオーバー

$("input[src*='_off.']").rollOver();
button要素内のimg要素をロールオーバー

$("button").rollOver( { find: "img" });

アルファロールオーバー ( alphaRo )

透明度の変化によるロールオーバーを設定する関数

関数概要

{target}.alphaRo({option});
{target}
適用する対象
{option}
[from](number) = 1
ロールオーバー前の透明度
[to](number) = 0.8
ロールオーバー後の透明度
[fade](boolean) = false
アニメーションの有無
[speed](*) = normal
アニメーションの速度
（slow, normal, fastの三段階、もしくはミリ秒）

デモ

スタンダード

$("a.alp").alphaRo();
フェードアニメーション

$("a.alpFade").alphaRo({fade:true});
アニメーション速度変更

$("a.alpSp").alphaRo({fade:true,speed:"fast"});
透明度変更

$("a.alpOpa").alphaRo({from:0.5,to:1,fade:true});

スムーススクロール ( smScroll )

ページ内リンクをスクロールによって移動する関数

関数概要

{target}.smScroll({option});
{target}
適用する対象
{option}
[speed](*) = 700
アニメーションの速度
（slow, normal, fastの三段階、もしくはミリ秒）
[easing](string) = null
アニメーションにかけるイージング名
イージング名一覧

デモ

スタンダード

▲▲▲ページトップへ（デモ）▲▲▲

$(".pnStd").smScroll();
速度変更

▲▲▲ページトップへ（デモ）▲▲▲

$(".pnSp").smScroll({speed:300});
イージング
※要easingプラグイン

▲▲▲ページトップへ（デモ）▲▲▲

$(".pnEase").smScroll({speed:1000,easing:"easeOutBounce"});
URL変更無し
※URL変更する際にページが崩れてしまうバグが発生する事があったための救済措置です、只今原因究明中

▼▼▼サブウィンドウ表示 ( anotherWin )へ（デモ）▼▼▼

サブウィンドウ表示 ( anotherWin )

サブウィンドウを表示する関数
ブラウザによってはリサイズは無効には出来ません

関数概要

{target}.anotherWin({option});
{target}
適用する対象
{option}
[width](number) = 500
サブウィンドウの横幅
[height](number) = null
サブウィンドウの縦幅、未入力の際は親ウィンドウに合わせる
[name](string) = subWindow
サブウィンドウ名
[scrollbars](string:yes/no) = yes
スクロールバーの有無
[resizable](string:yes/no) = yes
リサイズの可否
[menubar](string:yes/no) = yes
メニューバーの有無
[attr](boolean) = false
HTML側でサイズの指定をさせる（size="width,height"）
heightは省略可能、省略時は親ウィンドウに合わせる

デモ

スタンダード

サブウィンドウを開く

$(".sub").anotherWin();
全指定

サブウィンドウを開く

$(".subAll").anotherWin({width:300,height:300,name:"sub",scrollbars:"no",resizable:"no",menubar:"no"});
attrオプション

サブウィンドウを開く

HTML
<a class="subAttr" href="sub.html" data-size="500,500">サブウィンドウを開く</a>
JavaScript
$(".subAttr").anotherWin({attr:true});

範囲リンク ( setDivAnchor )

対象の要素をリンク要素と偽装し、包括するリンク要素の範囲を広げる

関数概要

{target}.setDivAnchor({option});
{target}
拡大したいリンク範囲
{option}
[not](string) = null
{target}の適用外指定
[class](string) = act
マウスオーバー時に{target}に付与するクラス名
[force](boolean) = true
window.open、window.locationを利用してページ遷移
ページ遷移をせずにアクションを行う場合はfalseにしておく
[over](function) = null
マウスオーバー時の動作拡張
[out](function) = null
マウスアウト時の動作拡張
[is_ro](boolean) = true
範囲内の画像をロールオーバーするか
[off](string) = '_off.'
ロールオーバー前画像の末尾語
[on](string) = '_on.'
ロールオーバー後画像の末尾語

デモ

スタンダード

テストリンク

CSS

.setDiv {
    padding:10px;
    width:200px;
    border:1px solid #ccc;
}
.act {
    background:#ccc;
}

JavaScript

$(".setDiv").setDivAnchor();
対象外設定

テストリンク

テストリンク対象外

テストリンク

CSS

.setDivNotTest {
    padding:10px;
    width:200px;
    border:1px solid #ccc;
}
.act {
    background:#ccc;
}

JavaScript

$(".setDivNotTest").setDivAnchor({not:".not"});

ストーカーメニュー ( stokerMenu )

スクロールに合わせて、少し遅れて付いてくるメニューを実装する

関数概要

{target}.stokerMenu({option});
{target}
付いてくるメニュー
{option}
[parent](string) = body
メニューの移動領域
[toPos](number/string:bottom) = 20
メニュー停止する位置（ウィンドウ上部から相対）
「b」を指定すると下付き、「rb」を指定すると右下付き
[delay](number) = 700
スクロール開始からメニューが動き始める時間
[speed](number) = 5
メニューの移動速度（値が大きいほうが遅い）
※ミリ秒等ではない

デモ

右下に出ているボタン

スタンダード

HTML

<div id="wrapper">
    <p id="topPage"><a href="#" class="pn">▲ページトップへ</a></p>
</div>

CSS

#topPage {
    margin:0;
    padding-bottom:15px;
}
    #topPage a {
        border:1px solid #fff;
        padding:2px 5px;
        color:#fff;
        display:block;
        background:#000;
        text-decoration:none;
        text-shadow:0 0 0;
    }
    #topPage a:hover {
        border:1px solid #eee;
        background:#222;
    }

JavaScript

$("#topPage").stokerMenu({parent:"#wrapper",toPos:"rb"});

例文表示 ( inputDefault )

placeHolderが実装されていないブラウザでも例文を表示させる
input / textareaのtitle属性が例文として扱われる

関数概要

{target}.inputDefault({option});
{target}
例文を表示するinput / textareaの指定
{option}
[form](string) = form
送信前に処理をする対象formの指定
デフォルトは全formが対象
[toColor](color) = #999999
例文表示時の文字色
[fromColor](color) = #333333
入力時の文字色

デモ

スタンダード

$(".inputDefault").inputDefault();
文字色変更

CSS

input.inputDefaultColor {
    background:#1a1a1a;
    color:#fff;
}

JavaScript

$(".inputDefaultColor").inputDefault({toColor:"#999999",fromColor:"#FFFFFF"});
